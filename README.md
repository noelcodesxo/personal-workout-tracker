# Personal Workout Tracker
This is a personal workout tracker that you can use to track your workouts, you can define any exercises you want to track, define any routines, and use it any time you want to record a workout.

The project has a server in Python.

## Server file structure
```
.
|
|—— server/
    |—— config/
    |—— api/ 
    |—— data/ 
    |—— utils/
    |—— __main__.py
|—— web/
    |—— index.ts
|—— README.md
```
